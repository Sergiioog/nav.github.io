World Wide (1990)

  • Primer navegador con interfaz gráfica.
  • Primer editor de HTML
  • Descontinuado en 1994
  • Creado por Tim Berners-Lee

Este navegador fue creado por Tim Berners-Lee en 1989 en el CERN (Conseil Européen de Recherche Nucléaire). Al año siguiente creó el primer navegador de la historia, el cuál se llamó WorldWideWeb aunque posteriormente cambió a Nexus.
En 1994 fue discontinuado, sin embargo, implementaba funciones esenciales hoy en día como los botones para navegar hacia adelante y hacia atrás. También fue el primer editor HTML además de ser capaz de utilizar diferentes protocolos como FTP, HTTP, NNTP y abrir archivos locales.

Viola (1992)

  • Considerado el primero de la historia.
  • Precursor del popular navegador Mosaic
  • Incluía scripts, CSS, HTML y XML
  • Desarrollado por Pei-Yuan Wei

Desarrollador por Pei-Yuan Wei, en la Universidad de California, se trata de uno de los navegadores mas famosos de la historia.
Trajo la inclusión de un lenguaje de scripts similar a Javascript, un modelo de objetos similar a Document Object Model (DOM), hojas de estilo en cascada similares a CSS, Client Side Includes y applets.

Mosaic (1993)

  • De software slibre pero sólo para sistemas Unix
  • Capacidad para acceder a servicios HTTP
  • Webs a color y con imágenes.
  • Creado por Marc Andreessen

Fue el primer navegador web en alcanzar popularidad entre el público en general y de los primeros con capacidad para mostrar gráficos junto al texto en línea.
A comienzos de la década de los 90 la web era principalmente texto. Se publicaban imágenes, fotografías o clips de audio o video, pero estas piezas de multimedia estaban ocultas detrás de los enlaces. Además, fue el primer navegador en dominar el mercado.

Lynx (1992-1993)

  • Basado en texto plano
  • Permitía protocolos HTTP,HTTPS y FTP entre otros
  • Las versiones actuales admiten SSL Y HTML pero no JavaScript
  • Desarrollador por Lou Montulli,Michael Grobe y Charles Rezac

Lynx fue diseñado para sistemas Unix y VMS, y se ejecuta en particular en GNU / Linux . Además se encuentran disponibles versiones para MS-DOS y para Windows .
Con su interfaz fácil de integrar en un sintetizador de voz, Lynx era muy popular entre las personas con discapacidad visual, pero la síntesis de voz mejorada redujo su interés por los ciegos.
Lynx todavía se utiliza como herramienta de desarrollo web, ya que permite al usuario consultar un documento de la misma forma que lo haría un rastreador.

Netscape (1994)

  • Primer navegador comercial.
  • Perdió toda su cuota de uso en favor de Internet Explorer
  • Introdujo un lenguaje de script en su versión 2
  • Desarrollador por Marc Andreessen (uno de los desarrolladores de Mosaic)

Netscape hizo gratuita la disponibilidad de su software porque tenía en sus políticas la noción de que el software para Internet no debía tener coste. En 1997, el Netscape Navigator 2.0 fue el primer navegador en incluir un lenguaje de script en las páginas web, al introducir JavaScript en su versión 2. Originalmente, apenas servía para algo más que para validar formularios, pero rápidamente se fue expandiendo.
Fue muy criticado por los partidarios de los estándares en Internet por introducir en el HTML gran cantidad de extensiones propietarias (o netscapismos), es decir, creadas por sus autores, sin respetar las recomendaciones del World Wide Web Consortium, lo que dañaba la compatibilidad de las páginas entre navegadores y al objetivo de llegar a la web semántica

Internet Explorer (1994)

  • Diseñado para posibilitar a los usuarios la navegación en las múltiples páginas web.
  • Brinda un soporte casi completo de HTML 4.01, CSS 1.0 y XML
  • Criticado debido a su limitado apoyo a estándares web abiertos
  • Desarrollador por Thomas Reardon

Internet Explorer introdujo una serie de extensiones propietarias de normas como HTML, CSS y DOM, lo cual dio lugar a la correcta visibilidad de algunas páginas web sólo a través del uso de este navegador. También incluyó una serie de suspensiones a Java Script, mismas que fueron adoptadas por otros navegadores. Algunas de ellas incluyen innerHTML, que retorna la cadena de HTML dentro de un elemento; el XML HTTP Request, el cual facilita el envío de la petición HTTP y la recepción de la recepción HTTP. Parte de estas funcionalidades no fueron posibles hasta la introducción de los métodos de DOM impulsados por W3C.

Opera (1995)

  • Era de tipo "trialware" (tenía que comprarse después de terminado el período de prueba)
  • En la versión 5.0 se financiaba con anuncios que se mostraban a los usuarios
  • Era portable, para lanzar el navegador como aplicación portátil
  • Desarrollado por la empresa noruega Opera Software

Opera es un navegador web compatible para ordenadores y para dispositivos móviles. Uno de los más rápidos que puedes instalar y que apuesta por la seguridad y la privacidad de sus usuarios pero también por una gran cantidad de herramientas y opciones sin necesidad de instalar extensiones, por ejemplo. Es gratuito y está disponible en todo el mundo con más de 50 idiomas compatibles.
Dispone de ventajas como una VPN incorporada o aplicaciones de mensajería que se pueden fijar en una zona del navegador mientras se usan otras webs.
